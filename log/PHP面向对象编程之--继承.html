继承：所谓继承就是 一个子类 通过 extends  父类  把父类的 (public / protected)属性和(public / protected)方法继承下来.
父类：把各个类中共有的属性和方法抽取出来形成一个父类，然后让其他类来继承该父类。
继承的基本语法是:
class 类名  extends 父类名{
        //在写自己需要的属性和方法.
}
代码实例：

<span style="font-size:18px;"><?php
///应用场景 开发一套学生管理系统(Pupil, Graduate , ...)
        //父类
        class Stu{
                public $name;
                protected $age;
                protected $grade;
        
                public function showInfo(){
                        
                        echo $this->name;
                }


        }
        //子类
        class Pupil extends Stu{
               
                public function testing(){
                        echo " 小学生考试..";
                }


        }


        class Graduate extends Stu{
               
                public function testing(){
                        echo "研究生考试..";
                }
        }


        //创建学生
        $stu1=new Pupil();
        $stu1->name="小明";
        $stu1->testing();
        //继承父类Stu
        $stu1->showInfo();
        //创建一个研究生
        $stu2=new Graduate();
        $stu2->name="老明";
        $stu2->testing();
        //继承父类Stu
        $stu2->showInfo();

?></span>

讨论继承细节: 

1.父类的 public  、protected 的属性和方法被继承. private 的属性和方法没有被继承，即私有属性和方法不能被继承.
2.一个类只能继承一个父类（指直接继承）.如果你希望继承多个类的属性和方法，则使用多层继承

代码实例：

<?php

class A{
    public $n1=90;
}
class B extends A{
    public $n2="hello";
}
class C extends B{
    
}
$c=new C();
echo "<br/>";
echo $c->n1."<br/>";
echo $c->n2."<br/>";
?>


当创建子类对象的时候，默认情况下，不会自动调用父类的构造方法. 

<span style="font-size:18px;"><?php
class A1{
    public $n1=90;
    //构造方法
    public function __construct(){
        echo "A1 __construct";
    }
}
class B1 extends A1{    
    function __construct(){
        echo "B1 __construct<br/>";
    }
}
//创建b对象实例，只调用子类构造方法，不会调用父类构造方法
$b=new B1();
?></span>

如果我们希望去调用父类的 构造方法，或者其它的方法(public /  protected) ,可以这样处理 (了解)
     类名::方法名()     或           parent::方法名()
代码实例：
<span style="font-size:18px;"><?php

class A2{
    public $n1=90;
    //构造方法
    public function __construct(){   
       echo "A2 __construct<br/>";
    }
    public function test(){
        echo "test()";
    }
}
class B2 extends A2{
    function __construct(){
        echo "B2 __construct<br>";
        //显式的调用父类的方法一
        A2::__construct();
        //显式的调用父类的方法二
        parent::__construct();
        //此方法也可调用普通函数
        parent::test();
        
    }
}
//创建b对象实例
$b=new B2(); 
?></span>

当一个子类的方法和父类的方法完全一样(public/protected), 我们称为 方法的覆盖 (方法重写)
