安装完成后配置密钥，配置过程如下：
配置git密钥：http://blog.csdn.net/u013372487/article/details/50825701
创建用户、邮箱：
在工作目录下,右键，点击TortoiseGit-->Setting,在打开的对话框里,左边选git,右边就会出现写用户名和邮箱的地方

输入用户名，邮箱设置成功。
若为远程版本库，git使用过程如下：
1、新建文件夹，右键 git clone

输入远程URL，点击 OK ，等待代码下载成功
2、修改代码，若添加了新的文件，需要 右键，点击TortoiseGit-->add,勾选要提交的文件，点击确定。
3、更新代码右键，点击TortoiseGit-->Pull
4、说明并提交代码右键，点击Git Commit ->"master"
输入程序修改说明，点击 commit & push
等待代码提交成功
5、冲突解决
有冲突的话,TortoiseGit-->Edit conflicts 比较之后解决冲突之后 TortoiseGit-->Resolve然后commit和push
