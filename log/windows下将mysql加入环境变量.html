1.    下载最新版mysql软件，将mysql安装到系统目录中，记录当前安装目录；

如安装mysql到C:/program files/mysql目录下

2.    打开win7系统——计算机——系统属性




3.    点击环境变量，进入环境变量设置菜单


4.    选中系统变量中的“path”，在path值开头处输入：

C:\program files\mysql\bin;

保存退出


5.    开始——搜索——CMD，打开CMD命令行，输入mysql -u 用户名 -p 登陆mysql

mysql -u 用户名：用户名处输入mysql的用户名

-p： 表示要输入密码
