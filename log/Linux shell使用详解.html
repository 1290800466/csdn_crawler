shell 用于解释用户输入的命令，将命令解释为二进制代码，并将结果返回到命令终端上。

语法：
文件 ： xxx.sh

#!/bin/bash
#comments,命令注释

具体命令
...
具体命令

执行 .sh 文件方式：

# sh xxx.sh

当xxx.sh文件有可执行权限时，可以用命令：

# ./xxx.sh

注意：
编写 shell 脚本时，可以将命令构造成函数，这样可以多次调用，如：

#!/bin/bash
function count() {
    ifconfig
}
count  #可多次调用

实例：

#!/bin/bash
/usr/bin/php /analysis2.com/cron/log_to_db.php >/dev/null 2>&1
/usr/bin/php /analysis2.com/cron/log_day_url.php >/dev/null 2>&1
/usr/bin/php /analysis2.com/cron/log_bot_day.php >/dev/null 2>&1
/usr/bin/php /analysis2.com/cron/day_url_leave.php >/dev/null 2>&1
/usr/bin/python /analysis2.com/cron/log_day_jump.py >/dev/null 2>&1
/usr/bin/php /analysis2.com/cron/url_pv.php >/dev/null 2>&1

上面语句执行是按顺序执行的！
