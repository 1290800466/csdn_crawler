windows系统SSHSecureShellClient与VMware Workstation虚拟机中Linux系统的连接步骤：

1、在windows系统中安装VMware Workstation虚拟机和SSHSecureShellClient客户端

2、在VMware Workstation虚拟机中安装Linux操作系统

3、打开Linux系统，将虚拟机“网络适配器”设置为 “NAT模式” （在此模式下，Linux系统IP网段和windows系统IP网段一致（要保证一致）），设备状态 勾选“已连接”和“启动时连接”，如下图所示：


4、打开Linux命令窗口，输入命令 ifconfig  查看系统 IP（Linux自动获取IP）

注意：若输入ifconfig  后没有显示IP，可关闭Linux系统，点击 “编辑”-->”虚拟网络编辑器“ -->点击”恢复默认设置“， 如下图所示     等恢复完成后，重新打开Linux操作系统，执行第3步


5、在windows平台下打开SSH Secure Shell Client客户端，点击”Quick Connect“  ，弹出”Connect to Remote Host“窗口，将获得的Linux系统IP、Linux账户输入下图输入框中，点击"Connect",弹出密码输入框，输入Linux密码，即可连接Linux系统。连接完成后即可将该客户端作为Linux命令窗口使用。


注意：若SSH Secure Shell Client客户端与Linux连接失败，打开Linux命令窗口，输入  service sshd restart     回车，重启sshd服务。

6、在windows平台下打开SSH Secure File Transfer Client客户端，点击”Quick Connect“  ，弹出”Connect to Remote Host“窗口，将获得的Linux系统IP、Linux账户输入下图输入框中，点击"Connect", 弹出密码输入框，输入Linux密码，即可连接Linux系统。连接完成后即可将该客户端作Linux系统与windows系统文件互传使用。
